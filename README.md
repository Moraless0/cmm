# FinPlan - Tu Plan Financiero 💵

Aplicación financiera integral desarrollada con HTML, CSS y JavaScript puro, diseñada para ayudar a gestionar tus finanzas personales mediante herramientas prácticas y elementos motivacionales.

## Descripción

FinPlan es una aplicación web responsive que combina herramientas financieras prácticas con un sistema de gamificación para motivar el desarrollo de hábitos financieros saludables.

### Características Principales

- **Dashboard**: Resumen financiero con estadísticas en tiempo real
- **Simulador de Ahorro**: Cálculos de interés compuesto con progreso mensual detallado
- **Bitácora Emocional**: Registro de reflexiones y emociones financieras con categorías y emojis
- **Metas Financieras**: Creación, seguimiento visual y cumplimiento de objetivos
- **Gráficos y Análisis**: Visualización del progreso y estadísticas detalladas
- **Gamificación**: Sistema de puntos, niveles (Novato → Experto) y logros desbloqueables
- **Configuración**: Personalización de tema, exportación de datos y gestión de almacenamiento

### Experiencia de Usuario

- **Responsive Design**: Optimizado para móvil y escritorio con sidebar adaptativo
- **Tema Claro/Oscuro**: Toggle con persistencia en localStorage
- **Navegación Intuitiva**: Sidebar lateral en escritorio, menú hamburguesa en móvil
- **Almacenamiento Local**: Todos los datos se guardan en el navegador (localStorage)

### Sistema de Gamificación

- **+10 puntos**: Entrada en bitácora o simulación guardada
- **+50 puntos**: Meta financiera creada
- **+100 puntos**: Meta completada

**Niveles**:
- Novato: 0-99 puntos
- Intermedio: 100-299 puntos
- Avanzado: 300-499 puntos
- Experto: 500+ puntos

### Tecnologías

- HTML5
- CSS3 con Variables CSS
- JavaScript ES6+
- localStorage para persistencia de datos
